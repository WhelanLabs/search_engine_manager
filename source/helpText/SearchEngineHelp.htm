Fields, Buttons, and Links

[Click HERE for a flash demo of the search engine management process.]

Fields

      Search Engine Port Number: Specifies the port number for which the search engine runs. Changes to this field require clicking of the update button and a restart of the search engine to become active.

      Organization Name: The name of the organization which maintains the search engine. This value is user to identify the crawler to web sites that it indexes. Changes to this value are committed by pressing the associated update button, and are used the next time the crawler runs.

      Organization URL: The URL of the organization which maintains the search engine. This value is user to identify the crawler to web sites that it indexes. Changes to this value are committed by pressing the associated update button, and are used the next time the crawler runs.

 

      Site Administrator E-Mail Address: The e-mail address of the administrator who maintains the search engine. This value is user to identify the crawler to web sites that it indexes. Changes to this value are committed by pressing the associated update button, and are used the next time the crawler runs.

Buttons

      (re)Start Search Engine: Stops the search engine (if running), then starts it. Restarting the search engine activates any committed configuration changes and loads the latest crawler results. The restart action might not be complete by the time the cursor returns from being an hourglass due to the asynchronous startup nature of the search engine.

      Stop Search Engine: Stops the search engine.

      Update Search Engine Port Number: Commits any changes to the search engine port number. The committed changes will be come active in subsequent invocations of the search engine.

      Update Administrative Information: Commits changes to the administrative information (Organization name, URL, and e-mail). The updated information is used in subsequent invocations of the crawler.

Links

      Launch search engine client: Opens the search engine home page in a browser window.

 

      View Search Engine Log File: Opens the search engine (Apache Tomcat) log file in a separate window.

 

      Help for this page: Opens this page in a browser window.
